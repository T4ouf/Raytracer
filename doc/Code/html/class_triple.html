Detailed Description

Triple class : Representation of a group of 3 doubles (without semantic) this class is providing a large toolkit for managing data with 3 doubles (operators, function, etc.) this class is then renamed regarding what the triple is (a Vector (3D), a Point (in a 3D space), a Color) and thus provide semantic
